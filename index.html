Install

The whole point: tell your agent “install theolundqvist/justshowme” and it sets itself up. Or do it by hand:

# 1. Copy the skill into your skills dir (the clone is just staging)
git clone https://github.com/theolundqvist/justshowme /tmp/justshowme
cp -r /tmp/justshowme/skills/visualize ~/.claude/skills/

# 2. PR screenshots (optional)
gh extension install theolundqvist/gh-img

# 3. Make it the default — paste docs/AGENTS-snippet.md
#    into your ~/.claude/CLAUDE.md (or AGENTS.md)

# 4. Drop the staging clone
rm -rf /tmp/justshowme
      
Works in any agent harness that reads a SKILL.md. Self-contained HTML output — no build step.
