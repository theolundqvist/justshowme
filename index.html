Just show me.

A Claude Code skill that makes your AI agent draw the answer — diagrams, animated flows, slide decks — instead of dumping markdown.

View on GitHub Install
Tell your agent:  “install justshowme”

Markdown is a wall. A diagram is a glance.

Past a hundred lines, a markdown explanation is something you skim and forget. The same idea as a drawing lands in two seconds. justshowme is opinionated that a colored card or a table is not a visual — a visual is a shape you understand by looking. So the agent draws the actual thing: boxes and arrows with spatial meaning, dots moving through a pipeline, a slider that moves a curve.

How you'll actually use it

You stop reading. You point at pictures and react.

Plan a feature

“Plan the rate limiter.” You get a slide deck — architecture, flow, the key decision — and iterate on a picture, not a 600-line plan.

Diagnose a bug

“Why is the data stale?” The root cause is drawn — write path, read path, the missing link in red. Grasp it instantly.

Open a PR with diagrams

Your agent attaches the 1–4 slides that explain the change, so reviewers see the architecture before reading the diff.

Six visual primitives, all generated HTML

No hand-drawing, no design tool — the agent composes these from a proven kit so they come out clean every time.

Install

The whole point: tell your agent “install justshowme” and it sets itself up. Or do it by hand:

# 1. Skill into your Claude Code skills dir
git clone https://github.com/theolundqvist/justshowme
cp -r justshowme/skills/visualize ~/.claude/skills/

# 2. PR screenshots (optional)
gh extension install theolundqvist/gh-img

# 3. Make it the default — paste docs/AGENTS-snippet.md
#    into your ~/.claude/CLAUDE.md (or AGENTS.md)

Works in any agent harness that reads a SKILL.md. Self-contained HTML output — no build step.
